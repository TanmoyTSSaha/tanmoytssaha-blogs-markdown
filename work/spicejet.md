## Overview

At SpiceJet, I worked as a data engineer on there big data infrastructure by optimising existing project and created a new nearly real-time data pipeline for analytics purpose.

---

## Key Responsibilities

- Designed and implemented a near real-time data pipeline.
- Worked on existing batch data processing pipeline which used to handle BAU tasks.
- Collaburated with stakeholders to develop data visualization dashboards using Django, HTML, TailwindCSS and JavaScript, providing actionable insights for decision-making.
- Collaburated with DBA team to develop a portal to monitor internal long running queries which hampers the regular activities.
- Implemented proper alerting mechanism.

---

## Key Achievements

- Designed and developed a near real-time data pipeline which pull it's data from emails when emails received and process it in near real-time and store the data in Iceberg table in parquet format.
- Implemented alerting mechanism for failure in data pipeline.
- Developed portal/dashboard to made the business decision easy for stakeholders. 
- Worked on improving the exising batch processing pipeline to make it metadata driven framework.

---

## Impact

- Improved decision making of stakeholders with near real-time data.
- Improved onboarding of BAU tasks from 1 week to 2-3 Days.
- Made the monitoring of long running resource heavy queries easy.
